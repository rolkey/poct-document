在临床检验的质量控制（质控）模块中，**室内质控**与**室间质控**的核心区别在于：**执行主体、目的、时间维度和评价标准**。

具体区别如下表所示：

| 维度 | 室内质控 (IQC) | 室间质控 (EQA，能力验证) |
| :--- | :--- | :--- |
| **执行主体** | 本实验室内部人员自行操作 | 外部机构（如临检中心）组织，多家实验室参与 |
| **核心目的** | 监测**本实验室**检测过程的稳定性、重复性；判断当批检测是否**在控**，能否发报告 | 评价**本实验室**与其他实验室结果的一致性；发现系统误差，评价**准确性** |
| **检测频率** | 高频：**随患者样本一同检测**（每批/每天/每项目） | 低频：按计划（通常**每年1-4次**，固定周期） |
| **质控品** | 实验室自备，通常为**稳定的、商品化**的质控品（分不同浓度水平） | 由组织方统一发放，**盲样**（实验室不知道靶值） |
| **评价依据** | 基于本实验室既往累积的**均值、标准差**（如Westgard规则） | 基于**所有参评实验室**的结果统计得出的**靶值**及允许范围 |
| **评价结果** | 在控 / 失控（若失控则需分析原因、纠正后再测患者样本） | 合格 / 不合格（通常以“能力验证得分”表示，如Z分数≤2） |
| **后果** | **直接影响当天患者报告能否发出** | **影响实验室资质、认可、评审**（不合格需整改，多次不合格吊销资格） |

### 通俗比喻理解
- **室内质控** = **日常自测**。好比运动员**每天**用同一把尺子量自己的身高，看测量过程是否稳定（是否手抖、尺子准不准）。
- **室间质控** = **统一考试**。好比多个考生做同一份**密封试卷**，比较大家的分数。看你是处于平均水平还是偏离太远。

### 补充要点
1.  **两者不可互相替代**  
    - 室内质控只能保证**重复性（精密度）**，但无法发现**系统偏差**（比如所有人的尺子都短了1cm，室内质控仍然“在控”）。  
    - 室间质控能发现**准确性**问题，但一年几次的低频检测无法监控日常漂移。

2.  **正确工作流程**  
    - **每日**：先做室内质控 → 在控 → 检测患者样本 → 发报告。  
    - **年度**：参加室间质评 → 回报结果 → 分析偏差 → 改进校准/流程。

3.  **软件模块中的体现**  
    - 室内质控模块：需要配置质控规则、质控品批号、累积均值、Levey-Jennings图、失控处理记录。  
    - 室间质控模块：需要录入组织方回报的靶值与范围、计算偏倚、生成能力验证总结报告。
