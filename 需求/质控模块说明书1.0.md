
# 6 质控管理模块（V2.0 完整版）
## 6.1 质控物（质控品）管理
1. 基础信息
   - 名称、项目、水平（低/中/高）
   - 生产厂家、批号、注册证
   - 靶值、SD、CV、允许范围
   - 失效日期、开瓶效期、储存温度

2. 入库与登记
   - 按科室/设备绑定
   - 批号唯一管理
   - 自动建档，不重复

3. 效期监控
   - 近效期提醒
   - 过期自动锁定
   - 禁止使用过期质控品

4. 与设备/项目绑定
   - 一个项目支持多水平
   - 一个设备支持多套质控
   - 执行质控自动匹配批号

5. 使用记录与追溯
   - 每次质控关联批号
   - 可按设备、人员、批号追溯
   - 使用量统计

## 6.2 质控规则管理（Westgard）
1. 内置规则库
   - 12s（警告）
   - 13s（失控）
   - 22s（失控）
   - R4s（失控）
   - 41s（失控）
   - 10x（失控）

2. 规则组合
   - 支持多规则组合
   - 按项目/设备独立配置
   - 可自定义规则

3. 参数配置
   - 均值、SD、CV
   - 允许偏倚、总误差

## 6.3 质控计划管理
1. 质控频率
   - 每日质控
   - 开机必做
   - 换批号必做
   - 维修校准后必做

2. 自动提醒
   - 未做质控提醒
   - 超期未做提醒
   - 消息推送责任人

## 6.4 质控执行与采集
1. 采集方式
   - 设备直连自动采集
   - 手工录入

2. 自动判定
   - 在控 / 警告 / 失控

3. 强制质控
   - 未质控不能检测患者样本
   - 报告无法发布

## 6.5 质控图（Levey-Jennings）
1. 自动生成
   - 均值线
   - ±2SD 警告线
   - ±3SD 失控线

2. 按水平分图
   - 水平1（低值）
   - 水平2（高值）

3. 导出/打印图表

## 6.6 失控处理闭环
1. 自动锁定
   - 失控→锁定设备→禁止发报告

2. 失控原因库
   - 设备故障
   - 定标漂移
   - 质控品变质
   - 试剂过期
   - 操作误差
   - 环境因素
   - 其他

3. 纠正措施
   - 重新定标
   - 更换质控品
   - 更换试剂
   - 重新检测
   - 工程师维修

4. 强制记录
   - 原因、措施、处理人、时间

5. 复查通过→解除锁定

## 6.7 一致性比对
1. 比对类型
   - POCT ↔ POCT 院内比对
   - POCT ↔ 中心实验室比对

2. 周期管理
   - 每6个月至少一次

3. 样本与靶值
   - 样本编号
   - 靶值、允许偏差

4. 结果判定
   - 合格/不合格
   - 偏倚计算
   - 合格率统计

5. 自动生成比对报告

## 6.8 室间质评（EQA）
1. 计划登记
2. 结果录入与上报
3. 回馈表上传
4. 成绩归档
5. 不合格整改记录

## 6.9 质控统计报表
1. 在控率、失控率
2. 月度质控报告
3. 设备质控合格率
4. 项目趋势分析
5. 导出PDF/Excel

## 6.10 权限与日志
1. 分级权限
   - 操作员、管理员、审核员
2. 禁止随意修改均值/SD
3. 全操作留痕日志
4. 符合三级等保
